1) софт работает на zenno poster 5, если у вас его нет то скачайте демо отсюда:
http://zennolab.com/ru/products/zennoposter/download/

2) запустите в zenno файл проекта Key_Rel_Check.xmlz

3) откройте входные настройки, впишите туда урл исследуемого сайта

4) откройте файл keywords.txt в папке проекта (там же где лежит Key_Rel_Check.xmlz) и впишите туда интересующие вас ключевые слова (каждое с новой строки)

5) добавьте проекту количество выполнений +1

6) если вам интересно как выполняется проект, нажмите "показать инстансы"

7) ждите завершения проекта

8) когда увидите опопвещение "Готово" и "Проект выполнен успешно" можете проверить файлы "relevant.txt" и "notrelevant.txt" в папке проекта (тамже где лежит Key_Rel_Check.xmlz) 
в "relevant.txt" будут ключи которым сайт релевантен, в "notrelevant.txt" будут те которым не релевантен.
